# Lab Work 6: Selenium WebDriver
Автоматизоване тестування UI за допомогою Selenium WebDriver (Java).

## Реалізовані сценарії (Таблиця 3.1):
1. **Сценарій №4:** Перевірка сортування товарів від дешевих до дорогих.
2. **Сценарій №3:** Додавання двох товарів до порівняння та перевірка переходу.

## Як запустити:
1. Відкрийте проект у **IntelliJ IDEA** як Maven Project.
2. Дочекайтеся завантаження залежностей `pom.xml`.
3. Запустіть клас `PnComUaTests` у папці `src/test/java`.
*(Завдяки `WebDriverManager` додатково завантажувати ChromeDriver не потрібно!)*
